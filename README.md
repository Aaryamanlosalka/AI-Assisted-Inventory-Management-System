# AI-Assisted Inventory and Product Information Management System for Steel Coil and Roofing Products

A web-based inventory management system developed as part of the B.Tech Internship at **Precichem India**. The application helps manage steel coil inventory, provides intelligent product search, recommendation features, and centralized product information through an easy-to-use web interface.

---

## Project Overview

This project was developed to simplify inventory management and improve access to product information within the steel processing industry. It enables users to upload inventory data, search products efficiently, view detailed stock information, and manage inventory through a centralized web application.

The system was developed using HTML, CSS, JavaScript, LocalStorage, and SheetJS.

## How to Run

1. Download or clone the repository.

```
git clone https://github.com/yourusername/AI-Assisted-Inventory-Management-System.git
```

2. Open the project folder.

3. Open **index.html** in any modern web browser.

No installation or server setup is required.

## Data Privacy

The inventory dataset used during development belongs to **Precichem India** and is not included in this repository due to confidentiality requirements.

Users may upload their own Excel file using the same format supported by the application.
